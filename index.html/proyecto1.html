Detalles del Proyecto: Tienda Online "TechGadget"

Este proyecto consistió en el desarrollo completo de una plataforma de comercio electrónico responsiva, diseñada para la venta de dispositivos electrónicos. El objetivo principal fue crear una experiencia de compra intuitiva y segura para los usuarios, desde la navegación de productos hasta el proceso de pago.

Características Principales:

  • Diseño Responsivo: Adaptable a cualquier dispositivo (escritorio, tablet, móvil).
  • Catálogo de Productos Dinámico: Permite añadir, editar y eliminar productos fácilmente.
  • Carrito de Compras: Funcionalidad completa para añadir, actualizar y eliminar ítems.
  • Proceso de Checkout Seguro: Simulación de pasarela de pagos integrada.
  • Filtros y Búsqueda Avanzada: Para una navegación eficiente del catálogo.
  • Panel de Administración: Gestión de pedidos, clientes y productos (funcionalidad no implementada en este prototipo, pero considerada en el diseño).

Tecnologías Utilizadas:

Este prototipo fue construido utilizando HTML5 para la estructura, CSS3 para el diseño y la responsividad, y JavaScript para la interactividad del carrito de compras y las validaciones de formulario. Se utilizaron bibliotecas como Font Awesome para los íconos.
