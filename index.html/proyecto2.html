Detalles del Proyecto: Aplicación de Lista de Tareas "MyTasks"

Esta aplicación web interactiva fue diseñada para ayudar a los usuarios a organizar y gestionar sus tareas diarias de manera eficiente. Permite añadir nuevas tareas, marcarlas como completadas y eliminarlas, todo ello con una interfaz de usuario limpia y responsiva.

Funcionalidades Clave:

  • Añadir Tareas: Ingresa nuevas tareas con facilidad.
  • Marcar como Completado: Alterna el estado de una tarea con un solo clic.
  • Eliminar Tareas: Remueve tareas ya no necesarias.
  • Persistencia de Datos: Las tareas se guardan en el almacenamiento local del navegador, persistiendo entre sesiones.
  • Interfaz Intuitiva: Diseño minimalista y fácil de usar.

Tecnologías Utilizadas:

Desarrollada íntegramente con HTML5 para la estructura, CSS3 para el estilizado y responsividad, y JavaScript puro para toda la lógica interactiva del lado del cliente. Se enfocó en la manipulación del DOM para una experiencia de usuario fluida sin recargas de página.
